# Market-Sutra
MarketSutra** is a real-time stock market simulator built with  Flutter and Flask. It shows live data from Google Finance, including stock prices, P/E ratios, market cap, and 25-week graphs. Features include Nifty/Sensex tickers, demo trading, PAN card OCR verification, face detection via CVZone, and live news from MoneyControl.
